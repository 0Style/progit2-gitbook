= 原书前言
欢迎阅读 Pro Git 第二版。

你可以在以下地址找到本书的在线版本： http://git-scm.com/book

与第一版类似，Pro Git 第二版以知识共享协议开源。

自第一版开源以来，许多事情都发生了变化。
其一，我们将本书的文本由 Markdown 迁移至 Asciidoc。
同时，我们还开始使用 O'Reilly 的 https://atlas.oreilly.com[Atlas平台] 持续构建生成本书，因此，所有主要格式在每种语言都是可用的。
